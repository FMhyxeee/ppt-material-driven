# PPT Final Checklist

发布前逐项勾选：

- [ ] 每页标题是否“结论先行”
- [ ] 每页是否只表达一个核心信息
- [ ] 关键数据是否标注来源与时间
- [ ] 图表单位、口径、时间维度是否一致
- [ ] 图片是否清晰且版权状态明确
- [ ] 字体层级是否统一（标题/正文/注释）
- [ ] 配色是否符合品牌规范（白底 + 深色文本 + 红色点睛）
- [ ] Logo 是否使用官方版本且未拉伸变形
- [ ] 动画与过渡是否克制且全篇统一
- [ ] 总页数与演讲时长是否匹配
- [ ] 商务场景是否给出明确行动项（Next Step）
- [ ] 学术场景是否给出证据链/局限/展望
