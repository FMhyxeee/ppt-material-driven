# Longi-Inspired 商务/学术风格规范

适用于参考“隆基绿能官网气质”的演示场景：白底、红色点睛、无衬线、信息清晰。

## 1) 风格定位

- 关键词：简约、科技、专业、克制
- 场景：商务汇报、学术演讲、路演、内部评审
- 原则：先清晰表达，再视觉增强

## 2) 配色规范

- 背景：白色或极浅灰
- 主文本：深灰或黑色
- 品牌强调色：`#E60012`（建议）
- 强调规则：每页 1 个主强调点，红色占视觉面积不超过 15%-20%

## 3) 字体规范

- 中文：微软雅黑 / 思源黑体（二选一）
- 英文：Arial / Helvetica / Open Sans（与中文风格匹配）
- 标题建议：正文的 1.5-2 倍
- 正文建议：不小于 16pt，推荐 18pt+
- 行距建议：1.2-1.35

## 4) 版式规范

- 一页一主题，标题结论先行
- 正文优先项目符号，避免大段文字
- 图文并排优先 40/60 或 50/50
- 叠字到图片时，必须保证对比度（遮罩或底块）
- 连续 3 页不得同一构图

## 5) 图表规范

- 关系到图表：
  - 对比 -> 柱状/对照表
  - 趋势 -> 折线/面积
  - 占比 -> 条形占比/环形（谨慎）
  - 流程 -> 流程图
- 去装饰：禁用 3D、重阴影、复杂纹理
- 强调方式：仅突出关键数据，其他元素灰阶弱化

## 6) 动画与过渡

- 全篇统一 1 套简洁过渡（淡入淡出/平滑）
- 内容动画仅用于引导阅读顺序
- 避免炫技动画，避免每页不同切换

## 7) 商务/学术差异化

- 商务：强调结论与行动项，适度视觉锚点（章节页、关键数字）
- 学术：强调方法、证据和可复核性，减少动画与装饰
- 商务页末建议有 `Next Step`，学术页末建议有 `局限/展望`

## 8) 合规与素材

- Logo 用官方版本，不拉伸变形
- 图片优先高清、可商用、可追溯
- 对外发布前补齐来源与许可说明
