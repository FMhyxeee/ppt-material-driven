# Corporate Minimal 样式规范

用于内部分享场景：克制、干净、信息密度适中，强调层级与排版。

## 1) 版心与网格

- 画布：16:9
- 页边距：左右 64，上下 48
- 栅格：12 列（双栏优先 5/7 或 4/8）

## 2) 字体体系

- 字体：微软雅黑 / 思源黑体 / 苹方（三选一，全套统一）
- H1：40-44（Semibold）
- H2：26-30（Semibold）
- 正文：18-20（Regular）
- 注释/页脚：12-14（Regular）
- 正文行距：1.2-1.35

## 3) 颜色体系（红白黑灰基线）

- 背景：白色或极浅灰（#FFFFFF / #F7F8FA）
- 主文本：深灰或黑色（#1F2329 / #111111）
- 强调色：隆基红（#E60012）
- 辅助灰阶：标题深灰、正文中灰、分割线浅灰
- 规则：每页最多 1 个强调色，红色视觉占比不超过 15%-20%

## 4) 形状与线条

- 圆角：8px（全局统一）
- 线条：1-1.5pt 浅灰
- 阴影：默认禁用；若使用，仅允许同一档极浅阴影

## 5) 常用 8 个版式

1. 封面：标题 + 副标题 + 右下角日期/部门
2. 目录：3-6 条，带章节编号
3. 章节页：大数字（01/02）+ 章节名
4. 左文右图：40/60（最高频）
5. 三卡片：原则/方法/结论
6. 对比页：现状 vs 方案（Before vs After）
7. 流程页：1→2→3→4（每步一句话）
8. 总结页：3 条 Takeaways + 下一步

## 6) 排版硬规则

- 全左对齐优先
- 间距只用 8/16/24/32
- 一页一个主题，标题写成结论句
- 卡片内边距 16-20
- 卡片间距 16-24
- 卡片高度尽量一致

## 7) 图表表达规则

- 保留：标题、关键数字、必要坐标
- 弱化：非关键信息（浅灰处理）
- 强调：单一强调色 + 注释箭头/标签
- 每页图表：最多 1 张主图 + 1 张辅助图
- 禁用：3D 图表、重阴影、复杂纹理

## 8) 样式规范页（建议放第2页或附录）

固定四行：

1. 字体：XX（标题半粗 / 正文常规）
2. 颜色：白底 / 深色文本 / 隆基红 / 灰阶
3. 圆角：8px；线条：1pt
4. 栅格：12列；边距：左右64上下48
